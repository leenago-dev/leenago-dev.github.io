Other Elements — abbr, sub, sup, kbd, mark

abbr

  • <abbr 사이에 뜻을 입력한 후, 닫아주면 hover로 full text가 뜨게 됩니다.

ex) GIF is a bitmap image format.

sub

  • <sub 를 사용하면, 아래첨자를 사용할 수 있습니다.

ex) H2O

sup

  • <sup 를 사용하면, 위첨자를 사용할 수 있습니다.

ex) Xn + Yn = Zn

kbd

  • <kbd 는 button type을 만들어주는 기능입니다.

ex) Press CTRL + ALT + Delete to end the session.

mark

  • <mark 기능은, highlight를 해주는 기능입니다.

ex) Most salamanders are nocturnal, and hunt for insects, worms, and other small creatures. 줄바꿈 테스트를 해보겠습니다. 이렇게 길게 적으면 과연 줄바꿈이 될는지, 제가 설정한 그 줄바꿈 기능이 작동할지 궁금하네요 호호호

Emoji can be enabled in a Hugo project in a number of ways.

The emojify function can be called directly in templates or Inline Shortcodes.

To enable emoji globally, set enableEmoji to true in your site’s configuration and then you can type emoji shorthand codes directly in content files; e.g.

🙈 :see_no_evil: 🙉 :hear_no_evil: 🙊 :speak_no_evil:

🫡

The Emoji cheat sheet is a useful reference for emoji shorthand codes.

Mathematical Notation

Mathematical notation in a Hugo project can be enabled by using third party JavaScript libraries.

In this example we will be using KaTeX

  • Create a partial under /layouts/partials/math.html
  • Within this partial reference the Auto-render Extension or host these scripts locally.
  • Include the partial in your templates like so:
1
2
3

{{ if or .Params.math .Site.Params.math }}
{{ partial "math.html" . }}
{{ end }}
  • To enable KaTeX globally set the parameter math to true in a project’s configuration
  • To enable KaTeX on a per page basis include the parameter math: true in content files

Note: Use the online reference of 1) Supported TeX Functions, 2) Supported TeX Functions

Examples

Inline math: $\varphi = \dfrac{1+\sqrt5}{2}= 1.6180339887…$

Block math: $$ \varphi = 1+\frac{1} {1+\frac{1} {1+\frac{1} {1+\cdots} } } $$
