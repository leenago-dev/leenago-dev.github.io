Part 1: API Key 발급받기

Step 1: 회원가입

Access Key 발급을 받기 위해선, 먼저 Unsplash Developer Page로 이동합니다.
화면 맨 우측 상단에 있는 Register as a developer를 눌러, 회원가입을 해줍니다.

Step 2: App Build 신청서 작성하기

  1. 회원가입 후, 노랑색으로 표시된 Your apps를 클릭합니다.
  1. 그럼 아래 화면처럼 뜰텐데요, New Application을 눌러줍니다.
  1. API Use and Guidelines에 모두 체크를 한 뒤, Accept terms를 클릭합니다.
  1. Application Information 팝업이 뜨면, 두 칸 모두 작성해줍니다.
    참고로, 저는 Personal blog 라고 입력했습니다!

Step 3: API Access Key 발급 완료

그럼, 이렇게 Personal Access Key 발급이 됩니다.
코드에서는 Access Key를 사용하면 됩니다.

Access Key를 외부에 공개하면 안된단 것, 잊지 마세요❗

Step 3: 함수 만들기 - params setting

이제 파라미터 setting을 해봅시다.
Get a random photo에서 사용할 수 있는 파라미터는 다음과 같습니다.

Param설명
collections특정 collection에 있는 사진 가지고 오기
topics특정 주제에 관련된 사진 가지고 오기
query특정 검색어에 관련된 사진 가지고 오기
count가지고 올 사진의 개수 (1~30 사이)

Part 3. 전체 코드 확인하기



import requests

def get_image_url(search_keyword, orientation):
  # Unsplash API 엔드포인트 및 API 키 설정
  api_endpoint = "https://api.unsplash.com/photos/random"
  access_key = YOUR ACCESS KEY # Unsplash에서 발급받은 API 키를 입력하세요.

  # 요청 매개변수 설정 (선택 사항)

  params = {
    "query": search_keyword,       # 검색 키워드 (예: nature)
    "orientation": orientation,    # 사진 방향 (예: landscape, portrait, squarish)
    "count": 1,                    # 가져올 이미지 개수
    "fm": "jpg"                    # 이미지 형식 요청
    }

    # API 요청 헤더 설정
  headers = {
      "Authorization": f"Client-ID {access_key}"
      }

  # API 요청 보내기
  response = requests.get(api_endpoint, headers=headers, params=params)

  # 응답 확인 및 이미지 URL 추출

  if response.status_code == 200:
    data = response.json()
    image_url = data[0]["urls"]["regular"] +"?fm=webp" # WebP 이미지 URL

    return image_url

  else:
    return ("Error fetching image:", response.status_code)
